## 사용 방법

### 방법 1 — Claude에서 직접 사용
각 `.md` 파일의 내용을 Claude 대화 시작 시 시스템 프롬프트로 붙여넣고 작업을 지시합니다.

```
[파일 내용 붙여넣기]

---
위 지침에 따라 다음 작업을 수행해줘:
[작업 내용]
```

### 방법 2 — GitHub Pages 배포 후 링크로 활용
이 레포지토리를 클론하고 GitHub Pages로 배포하면
각 담당자 지침을 웹에서 바로 확인할 수 있습니다.

### 방법 3 — API 연동
각 `.md` 파일을 Claude API의 system 파라미터로 전달해
담당자별 전용 AI 어시스턴트를 구성할 수 있습니다.

```python
import anthropic

with open("01_콘텐츠기획담당.md", "r") as f:
    system_prompt = f.read()

client = anthropic.Anthropic()
message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    system=system_prompt,
    messages=[{"role": "user", "content": "AI 강의 커리큘럼 만들어줘"}]
)
```
